# Почему не открывается http://localhost:3000

## Главное

**localhost:3000 открывается только когда на вашем компьютере запущен сервер разработки (Vite).**

- Если вы **просто открыли в браузере** http://localhost:3000 и не запускали сервер — браузер покажет ошибку («Не удаётся подключиться» / «Connection refused»). Так и должно быть: сервера нет.
- Адрес **начинает работать** только после того, как вы в терминале выполнили команду запуска и в консоли появилось что-то вроде:  
  `Local: http://localhost:3000/`

---

## Что нужно сделать по шагам

### 1. Откройте терминал в папке проекта

- В Cursor: Terminal → New Terminal (или `` Ctrl+` ``).
- Либо CMD/PowerShell: перейдите в папку проекта:
  ```bat
  cd f:\WebGamePrototype
  ```

### 2. Запустите сервер

В этом же терминале выполните:

```bat
npm run dev
```

**Важно:** окно терминала не закрывайте — пока команда выполняется, сервер работает и localhost:3000 будет открываться.

### 3. Посмотрите вывод в терминале

- **Если всё ок** — появится что-то вроде:
  ```
  VITE v4.x.x  ready in xxx ms
  ➜  Local:   http://localhost:3000/
  ```
  Тогда откройте в браузере именно этот адрес: http://localhost:3000/

- **Если порт 3000 занят** — Vite напишет, что использует другой порт, например:
  ```
  ➜  Local:   http://localhost:3001/
  ```
  Открывать нужно тот адрес, который показан в терминале (например http://localhost:3001/).

- **Если появилась ошибка** — скопируйте её текст полностью. По нему можно понять, почему не открывается (например, нет зависимостей, ошибка в коде или с Node.js).

---

## Частые причины «не открывается»

| Ситуация | Что делать |
|----------|------------|
| В браузере «Не удаётся подключиться» | Сначала запустите `npm run dev` в терминале и дождитесь сообщения про `Local: http://localhost:...` |
| Команда `npm run dev` не найдена / не работает | Выполните в папке проекта `npm install`, затем снова `npm run dev` |
| Ошибка про `rollup/parseAst` или модули | Удалите папку `node_modules` и файл `package-lock.json`, выполните `npm install`, затем `npm run dev` |
| Пишет, что порт 3000 занят | Закройте другое приложение на 3000 или откройте тот URL, который Vite покажет (например 3001) |
| Страница открылась, но белая / с ошибкой | Откройте в браузере DevTools (F12) → вкладка Console и посмотрите текст ошибки |

---

## Кратко

1. **Запуск сервера** — в папке проекта в терминале: `npm run dev`.
2. **Адрес** — тот, что выведен в терминале (обычно http://localhost:3000/).
3. **Ошибки** — смотреть в том же терминале и в консоли браузера (F12).

Без работающей команды `npm run dev` страница http://localhost:3000/ открываться не будет.
